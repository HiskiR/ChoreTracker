# ChoreTracker

## Aihekuvaus

Sovelluksen avulla pidetään kirjaa eri ryhmien, kuten kotitalousten askareista. Kirjatun tiedon avulla voidaan tarkastella mitä askareita kukin ryhmän jäsen tekee milloinkin ja kuinka paljon.
